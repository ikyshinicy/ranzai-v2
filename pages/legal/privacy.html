Kebijakan Privasi RanzAI

Terakhir diperbarui: 4 Juni 2026

RanzAI adalah web app kreatif yang menyediakan beberapa tools berbasis Engine RanzAI, seperti image-to-prompt, design brief, image-to-text, file converter, background remover, mockup, dan fitur lain yang masih terus dikembangkan.

1. Informasi yang Kami Kumpulkan

Kami dapat mengumpulkan informasi akun seperti email, nama profil dari login Google, dan data penggunaan dasar untuk menjalankan layanan. Jika pengguna mengunggah gambar atau file ke tools RanzAI, data tersebut digunakan untuk memproses permintaan pengguna.

2. Penggunaan Informasi

  • Memproses login dan autentikasi pengguna.
  • Menjalankan tools yang dipilih pengguna.
  • Menampilkan dashboard, kredit, dan riwayat penggunaan.
  • Meningkatkan stabilitas dan kualitas layanan RanzAI.

3. File dan Gambar yang Diunggah

File atau gambar yang diunggah digunakan untuk proses sesuai tool yang dipilih. Pengguna bertanggung jawab memastikan file yang diunggah tidak melanggar hak cipta, privasi orang lain, atau hukum yang berlaku.

4. Layanan Pihak Ketiga

RanzAI dapat menggunakan layanan pihak ketiga seperti Supabase untuk autentikasi dan penyimpanan, Google OAuth untuk login, serta layanan AI/API lain untuk menjalankan fitur tertentu.

5. Keamanan Data

Kami berupaya menjaga keamanan data pengguna dengan cara yang wajar. Namun, tidak ada sistem internet yang sepenuhnya bebas risiko.

6. Hak Pengguna

Pengguna dapat berhenti menggunakan layanan kapan saja. Jika ingin meminta penghapusan data akun atau menanyakan privasi, hubungi kami melalui email resmi pengelola.

7. Perubahan Kebijakan

Kebijakan ini dapat diperbarui sewaktu-waktu mengikuti perkembangan fitur RanzAI. Versi terbaru akan ditampilkan di halaman ini.
